Vegetables in Cylinders

Information

  • Content: Vegetables in Cylinders
  • Size: 94 x 31.5 cm
  • Year:

    • Description

  • Vegetables are one of the things people treat without respect. True. They are the cheapest in the market; no one seems to value these daily foods. Nonetheless, I see unique beauty and traits in vegetables. Wanting people to recognize this value, I created this art piece, in which I put different kinds of vegetables in transparent cylinders. Placed in cylinders, vegetables appear to be a lot more colorful, unique, and beautiful. Value can be created when one presents the same things in different manners.
